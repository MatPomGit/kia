Interaktywny terminal Bash

Samodzielne środowisko ćwiczeniowe działające w przeglądarce. Terminal emuluje podstawowe polecenia Bash i korzysta z wirtualnego systemu plików, dlatego nie ma dostępu do rzeczywistych plików ani systemu operacyjnego komputera.

student@ndp: wirtualny Bash

Obsługiwane polecenia

Kliknij polecenie, aby wstawić je do terminala.

pwd ls -la cd mkdir -p touch cat echo printf cp mv rm rmdir tree find file stat du head tail tac nl wc grep -n sort uniq -c cut tr sed tee diff basename dirname seq env printenv export unset which type test -f date whoami hostname history man help

Ćwiczenia

1. Struktura katalogów podstawowe

Utwórz katalog projekt z podkatalogami src, docs i tests.

2. Pliki tekstowe podstawowe

Utwórz plik projekt/docs/readme.txt i zapisz w nim krótki opis.

3. Przetwarzanie tekstu średnie

Wyświetl tylko wiersze zawierające słowo bash.

4. Potok średnie

Posortuj wiersze i policz liczbę powtórzeń.

5. Wyszukiwanie plików średnie

Znajdź wszystkie pliki tekstowe w katalogu domowym.

6. Zmienne środowiskowe średnie

Utwórz zmienną PROJEKT i wyświetl jej wartość.

7. Przetwarzanie kolumn zaawansowane

Wyświetl pierwszą kolumnę pliku rozdzielanego dwukropkami.

Najważniejsze zasady

  • Tab uzupełnia nazwy plików.
  • ↑ i ↓ przeglądają historię.
  • Ctrl+L czyści ekran.
  • Ctrl+C przerywa bieżące wprowadzanie.
  • Obsługiwane są operatory >, >> i |.
  • Można definiować zmienne: NAZWA=wartość.
  • Dostępne są zmienne $HOME, $USER, $PWD i $?.
  • Część poleceń obsługuje najczęściej używane opcje, np. grep -inv, sort -rn i uniq -c.

To środowisko jest emulatorem edukacyjnym, nie pełną implementacją Bash. Nie uruchamia programów systemowych, skryptów binarnych ani poleceń wymagających dostępu do systemu operacyjnego. Jego zaletą jest bezpieczeństwo i możliwość publikacji jako pojedynczy plik HTML w GitHub Pages.
